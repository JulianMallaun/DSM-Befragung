# DSM Befragungs-App (Hotel) – v20

Änderungen:
- **Bereichsbezeichnung unter Geräten ohne Präfix** („A) “, „B) “, „C) “ werden entfernt), z. B. unter „Fritteuse“ steht nur noch **Küche**.
